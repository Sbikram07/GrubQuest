# 🍔 GrubQuest - Food Delivery Web App

GrubQuest is a modern and responsive food delivery web app built using **React**, **Tailwind CSS**, and **Vite**. It features a restaurant listing, search, carousel sliders, and detailed food menus—all powered by local JSON data (no backend).

![GrubQuest Screenshot](https://via.placeholder.com/1200x600?text=GrubQuest+Screenshot)

## 🚀 Features

- 🏪 Browse restaurants with modern cards and carousels
- 🍽 View detailed food items by cuisine type
- 📱 Fully responsive UI (mobile-first)
- 🔁 Image fallback and shimmer loading for better UX
- 💡 About and Contact pages included
- 🔎 Search functionality for restaurants
- 🎨 Tailwind CSS styling with custom configs
- ⚡ Fast build with Vite

## 🛠 Tech Stack

- **React** + **Vite**
- **Tailwind CSS**
- **Keen Slider** (for carousels)
- **Local JSON data** (no backend required)


## 📦 Installation & Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/Sbikram07/GrubQuest.git

# 2. Install dependencies
